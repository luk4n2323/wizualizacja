
## Render deployment

This repository includes `render.yaml` for a Render Static Site.

Recommended setup:

1. Push this repository to GitHub.
2. In Render, create a new Blueprint or Static Site from the GitHub repo.
3. Render will publish `./outputs`.
4. Build command: `echo "Static site - no build step"`.

The deployed client link will be available as an `onrender.com` URL.
